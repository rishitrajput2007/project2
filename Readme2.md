# 🚀 Logic Box – Pattern Generator & Number Analyzer

Welcome to **Logic Box**, a beginner-friendly Python project that combines creativity with number analysis using loops, conditions, and match-case statements.

---

# ✨ Features

## 🎨 Pattern Generator

Generate star (`*`) patterns dynamically based on user input.

Example:

```python
*
* *
* * *
* * * *
```

---

## 🔢 Number Analyzer

Analyze a range of numbers and identify whether each number is:

* Even
* Odd

The program also calculates the total sum of all numbers in the given range.
